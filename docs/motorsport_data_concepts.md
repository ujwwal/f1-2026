# Motorsport Data Concepts

## Lap-level metrics

- **lap time**: total time for one lap
- **sector times**: split timing checkpoints (S1, S2, S3)
- **stint**: consecutive laps on one tyre set
- **compound**: tyre type (e.g., soft/medium/hard)
- **tyre life**: number of laps completed on current tyre set
- **track status**: conditions/flags that affect lap performance

## Strategy context

Race strategy decisions depend on:
- pace consistency
- tyre degradation
- pit stop timing windows
- race interruptions and track status changes

## Why this matters for analytics and ML

These fields become the foundation for:
- pace trend analysis
- degradation curves
- pit strategy evaluation
- later predictive models
